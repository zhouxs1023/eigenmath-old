1) Compute the confidence interval for a mean, variance is known.
See lecture042, p. 9.
$$\sigma=1.5$$
$$n=64$$
$$\bar Y=16.4$$
$$1-\alpha=0.95$$
$$Z_{\alpha/2}=1.96$$
The confidence interval is
$$\bar Y\pm Z_{\alpha/2}\sigma/\sqrt n=16.4\pm0.3675
=(16.03,\,16.77)$$

7) Compute the confidence interval for a difference of proportion.
See lecture045, p. 17.
$$n=500$$
$$X=178$$
$$\hat p_X=X/n=0.356$$
$$m=450$$
$$Y=220$$
$$\hat p_Y=Y/m=0.4889$$
$$1-\alpha=0.95$$
$$Z_{\alpha/2}=1.96$$
$$Z_{\alpha/2}\sqrt{\hat p_X(1-\hat p_X)/n+\hat p_Y(1-\hat p_Y)/m}
=0.0624$$
The confidence interval is
$$(\hat p_X-\hat p_Y-0.03184,\,\hat p_X-\hat p_Y+0.03184)
=(-42.06,\,-41.94)$$

9b) Compute the confidence interval for a variance.
See lecture043, p. 6.
$$n=9$$
$$S=0.7219$$
$$1-\alpha=0.90$$
$$\alpha/2=0.05$$
$$\chi^2_{n-1,\alpha/2}=\chi^2_{8,0.05}=15.51$$
$$\chi^2_{n-1,1-\alpha/2}=\chi^2_{8,0.95}=2.733$$
The confidence interval is
$$((n-1)S^2/15.51,\,(n-1)S^2/2.733)=(0.2688,\,1.5255)$$

10) Compute a confidence interval for difference in means,
both variances are unknown but equal.
See lecture045, p. 4.
Mean and standard deviations from Excel.
$$n=12$$
$$\bar X=10.375$$
$$S_X=1.4$$
$$m=11$$
$$\bar Y=12$$
$$S_Y=1.612$$
$$1-\alpha=0.95$$
$$\alpha/2=0.025$$
$$S_p=\sqrt{((n-1)S_X^2+(m-1)S_Y^2)/(n+m-2)}=2.264$$
$$t_{n-1,\alpha/2}=t_{11,0.025}=2.201$$
$$t_{n-1,\alpha/2}S_p\sqrt{1/n+1/m}=(2.201)(2.264)(0.4174)=2.08$$
The confidence interval is
$$\bar X-\bar Y-2.08,\,\bar X-\bar Y+2.08)=(-3.705,\,0.455)$$

\vfill
\eject
