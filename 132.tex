2. Let random variable $X$ be uniformly distributed on the interval $(0,1)$.
Find the probability density function of $Y$, if $Y=\ln{X\over1-X}$.

\bigskip
We have
$$Y=F^{-1}(X)=\ln{X\over1-X}$$
(Note the flip $X\leftrightarrow Y$ compared to the theorem in Note 4.)
We have $F^{-1}$ but we need $F$.
$$y=\ln{x\over1-x}
,\qquad
e^y={x\over1-x}
,\qquad
e^{-y}={1-x\over x}={1\over x}-1
,\qquad
x={1\over1+e^{-y}}
$$
Hence
$$X=F(Y)={1\over1+e^{-Y}}$$
Now we can take the derivative of $F$ to get the probability distribution function
for $Y$.
$$f(y)={dF\over dy}={e^{-y}\over(1+e^{-y})^2}$$
