{\bf Trinomial Tricks}

\bigskip
1. Example
$$p(x,y)=C(20,x)\times C(20-x,y)\times 0.2^x\times 0.3^y\times (1-0.2-0.3)^{20-x-y}$$
$$(X,Y)\sim Trinomial(n{=}20,\,p_X{=}0.2,\,p_Y{=}0.3)$$
$$Cov(X,Y)=-np_Xp_Y=-1.2$$
$$X\sim Binomial(n{=}20,p{=}0.2),\qquad E(X)=np=4$$
$$Y\sim Binomial(n{=}20,p{=}0.3),\qquad E(Y)=np=6$$

\bigskip
2. Example. What is the value of $Var(Y|X{=}12)$?
\medskip
Solution: Don't panic! First, just work out the probability mass function.
$$p(y)={p(12,y)\over p_X(12)}$$
The trick here is that $p_X$ is a binomial distribution so we can put
$$p(y)={C(20,12)\times C(8,y)\times 0.2^{12}\times 0.3^y\times 0.5^{8-y}
\over C(20,12)\times 0.2^{12}\times (1-0.2)^8}
={C(8,y)\times 0.3^y\times 0.5^{8-y}\over 0.8^8}$$
At first it looks like we're stuck.
However, note that $0.8^8=0.8^y\times 0.8^{8-y}$ so we can put
$$p(y)=C(8,y)\times(3/8)^y\times (5/8)^{8-y}$$
which is a binomial distribution with $n=8$ and $p=3/8$. Hence
$$Var(Y|X=12)=np(1-p)=1.875$$
