3. Assume random variable $\chi_n^2$ has a Chi-square distribution
with $n$ degrees of freedom.

\bigskip
a) $P\{\chi_{10}^2>C\}=0.01$, find $C$.

\medskip
From Table 8, row 10, column $.01$, the solution is $C=23.21$.

\bigskip
b) $P\{\chi_{15}^2<C\}=0.99$, find $C$.

\medskip
Use row 15 and, per lecture 34, column $1-0.99=0.01$.
Hence $C=30.58$.

\bigskip
c) $P\{\chi_{21}^2>C\}=0.95$, find $C$.

\medskip
From Table 8, row 21, column $0.95$, $C=11.59.$
