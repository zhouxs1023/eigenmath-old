Show using $\epsilon$-$\delta$ that any function $f$ with domain $Z$ is
continuous at every point.

\bigskip
Let $\delta=1$.
Then $|x-c|<\delta$ implies $x=c$ because the domain is $Z$.
For $x=c$ we have $|f(x)-f(c)|=0$ hence $|f(x)-f(c)|<\epsilon$
for all $\epsilon>0$.

\end
