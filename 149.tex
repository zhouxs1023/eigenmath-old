\beginsection 6.

Consider the random variables $X$ and $Y$ with joint probability
density function as
$$f(x,y)=\cases{
\displaystyle{3x^2\over y^3},&for $0\le x\le y\le1$\cr
\cr
0,&otherwise\cr
}$$

\beginsection 6. (a)

Find the value of $P(Y\le0.5)$.
\medskip\noindent
Solution: First, sum over $x$ from 0 to $y$.
Then sum over $y$ from 0 to 0.5.
$$\eqalign{
P(Y\le0.5)
=\int_0^{0.5}\int_0^y{3x^2\over y^3}\,dx\,dy
=\int_0^{0.5}\left({x^3\over y^3}\bigg|_0^y\right)\,dy
=\int_0^{0.5}\,dy
=y\bigg|_0^{0.5}
=0.5\cr
}$$

\beginsection 6. (b)

Find the value of $P(X+Y\le 1)$.
\medskip\noindent
Solution: See lecture 55 slide 7.
$$\eqalign{
P(X+Y\le1)
&=\int_0^{1/2}\int_0^y{3x^2\over y^3}\,dx\,dy+
\int_{1/2}^1\int_0^{1-y}{3x^2\over y^3}\,dx\,dy\cr
&=\int_0^{1/2}\left({x^3\over y^3}\bigg|_0^y\right)\,dy+
\int_{1/2}^1\left({x^3\over y^3}\bigg|_0^{1-y}\right)\,dy\cr
&=\int_0^{1/2}dy+\int_{1/2}^1\left({1\over y^3}-{3\over y^2}+{3\over y}-1\right)\,dy\cr
&=y\bigg|_0^{1/2}+\left(-{1\over2y^2}+{3\over y}+3\ln y-y\right)\bigg|_{1/2}^1\cr
&=3\ln2-{3\over2}
}$$

\beginsection 6. (c)

Find the marginal p.d.f. of $Y$.
\medskip\noindent
Solution: See lecture 55 slide 10.
$$f_Y(y)=\int_0^y{3x^2\over y^3}\,dx={x^3\over y^3}\bigg|_0^y=1$$

\beginsection 6. (d)

Find the conditional p.d.f. of $Y$ given $X=0.5$.
\medskip\noindent
Solution: See lecture 55 slide 17.
$$f_X(x)=\int_x^1{3x^2\over y^3}\,dy=-{3x^2\over2y^2}\bigg|_{y=x}^{y=1}
={3\over2}-{3\over2}x^2$$
$$f_{Y|X{=}0.5}(y)={f(x,y)\over f_X(x)}\bigg|_{x=0.5}={3x^2/y^3\over3(1-x^2)/2}\bigg|_{x=0.5}
={2\over3y^3}$$

\beginsection 6. (e)

Find the value of $Var(X|Y{=}0.5)$.
\medskip\noindent
Solution:
$$f_{X|Y{=}0.5}(x)={f(x,y)\over f_Y(y)}\bigg|_{y=0.5}={3x^2/y^3\over1}\bigg|_{y=0.5}
=24x^2$$
$$E[X|Y{=}0.5]=\int_0^{0.5}24x^3\,dx=6x^4\bigg|_0^{0.5}={3\over8}$$
$$E[X^2|Y{=}0.5]=\int_0^{0.5}24x^4\,dx={24\over5}x^5\bigg|_0^{0.5}={3\over20}$$
$$Var(X|Y{=}0.5)=E[X^2|Y{=}0.5]-(E[X|Y{=}0.5])^2=(3/20)-(3/8)^2=3/320=0.0094$$
