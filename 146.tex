\beginsection 3.

Consider the random variables $X$ and $Y$ with j.p.m.f. as
$$p(x,y)={}_{10}C_x{}_{10-x}C_y0.4^x0.3^{10-x},\qquad
x=0,1,\ldots,10,\quad y=0,1,\ldots,10,\quad x+y\le10$$

\beginsection 3. (a)

Find the marginal p.m.f. of $Y$.
\medskip\noindent
Solution: Sum over $X$ for each $Y$.
$$p_Y(y)=\sum_{x=0}^{10-y}p(x,y)=\cases{
282475249/10000000000,&$y=0$\cr
121060821/1000000000,&$y=1$\cr
466948881/2000000000,&$y=2$\cr
66706983/250000000,&$y=3$\cr
200120949/1000000000,&$y=4$\cr
257298363/2500000000,&$y=5$\cr
36756909/1000000000,&$y=6$\cr
2250423/250000000,&$y=7$\cr
2893401/2000000000,&$y=8$\cr
137781/1000000000,&$y=9$\cr
59049/10000000000,&$y=10$\cr
}$$

\beginsection 3. (b)

Find the value of the second moment $X$.
\medskip\noindent
Solution: Use a computer.
$$M(t)=E[e^{tX}]=\sum_{x,y}e^{tx}p(x,y)$$
$${\partial^2M\over\partial t^2}\bigg|_{t=0}={92\over5}=18.4$$

\beginsection 3. (c)

Find the value of $P(X+Y=3)$.
\medskip\noindent
Solution: Sum over all $p(x,y)$ for which $x+y=3$.
$$P(X+Y=3)=p(0,3)+p(1,2)+p(2,1)+p(3,0)={2{,}250{,}423\over250{,}000{,}000}=0.0090$$

\beginsection 3. (d)

Find the m.g.f. of $Y$ given $X=2$.
\medskip\noindent
Solution:
$$M(t)=E[e^{tY}]=\sum_{y=0}^{10}e^{ty}\left[{p(2,y)\over p_X(2)}\right]$$

\beginsection 3. (e)

Find the value of $P(Y{\le}3|X{=}2)$.
\medskip\noindent
Solution: Use a computer.
$$P(Y{\le}3|X{=}2)={p(2,0)+p(2,1)+p(2,2)+p(2,3)\over p_X(2)}=0.3633$$
