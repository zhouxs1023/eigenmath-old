Find the inverse of the following matrix.
$$A=\pmatrix{1&2\cr3&4}$$

\bigskip
Solution: Apply elementary row operations that turn $A$ into $I$,
like this:
$$E_3E_2E_1A=I$$
Multiply both sides of this equation by $A^{-1}$ to prove to yourself that
$$E_3E_2E_1=A^{-1}$$
Use an augmented matrix to apply row operations
to both $A$ and $I$
simultaneously.

$$\pmatrix{
1&2&:&1&0\cr
3&4&:&0&1\cr
}$$

$R_2-3R_1\rightarrow R_2$

$$\pmatrix{
1&2&:&1&0\cr
0&-2&:&-3&1\cr
}$$

$R_1+R_2\rightarrow R_1$

$$\pmatrix{
1&0&:&-2&1\cr
0&-2&:&-3&1\cr
}$$

$-R_2/2\rightarrow R_2$

$$\pmatrix{
1&0&:&-2&1\cr
0&1&:&3/2&-1/2\cr
}$$

The result is

$$A^{-1}=\pmatrix{-2&1\cr3/2&-1/2}$$

Verify using the exchange trick:

$$-{1\over2}\pmatrix{4&-2\cr-3&1}
=\pmatrix{-2&1\cr3/2&-1/2}$$

\end
