{\bf Problem A.} True or False:
Let $x,y\in Z$. If $x<y$, then $-3y<x$.
\bigskip
False by the counter example $x=-2$, $y=-1$ which leads to
the false statement $3<-2$.

\bigskip
{\bf Problem B.}
True or False: Let $x,y\in Z$.
If $21x<21y$, then $x<y$.
\bigskip
True by the following.

{\bf Proof.}
Let $x,y\in Z$, $21x<21y$. Then

$21y-21x\in N$ by the definition of less-than (Definition 2.3).

$21(y-x)\in N$ by the distributive law.

$y-x\in N$ by Proposition 2.2 and $21\in N$.

$x<y$ by the definition of less-than.

\bigskip
{\bf Problem C.} Prove the following lemma.
\medskip
{\bf Lemma.} Let $a,b\in Z$. If $-b\in N$, the $a+b<a$.
\bigskip
{\bf Proof.} Let $a,b\in Z$, $-b\in N$. Then

$0+(-b)\in N$ by Axiom II, additive identity.

$a+(-a)+(-b)\in N$ by Axiom III, additive inverse.

$a+(-1)a+(-1)b\in N$ by Lemma 1.4, $(-1)a=-a$.

$a+(-1)(a+b)\in N$ by the distributive law.

$a-(a+b)\in N$ by Lemma 1.4.

$a+b<a$ by the definition of less-than (Definition 2.3).
