\bigskip
{\bf Problem A.} True or False:
Let $x,y\in Z$. If $x<y$, then $-3y<x$.
\bigskip
False by the counter example $x=-2$, $y=-1$ which leads to
the false statement $3<-2$.

\bigskip
\bigskip
{\bf Problem B.}
True or False: Let $x,y\in Z$.
If $21x<21y$, then $x<y$.
\medskip
True.
\bigskip
{\bf Proof.}
Let $x,y\in Z$, $21x<21y$. Then
$$\eqalign{
21y-21x\in N		&\qquad\hbox{definition of less-than}\cr
21(y-x)\in N		&\qquad\hbox{distributive law}\cr
y-x\in N		&\qquad\hbox{Proposition 2.2 and $21\in N$}\cr
x<y			&\qquad\hbox{definition of less-than}\cr
}$$

\bigskip
{\bf Problem C.} Prove the following lemma.
\medskip
{\bf Lemma.} Let $a,b\in Z$. If $-b\in N$, the $a+b<a$.
\bigskip
{\bf Proof.} Let $a,b\in Z$, $-b\in N$. Then
$$\eqalign{
0+(-b)\in N		&\qquad\hbox{additive identity}\cr
a+(-a)+(-b)\in N	&\qquad\hbox{additive inverse}\cr
a+(-1)a+(-1)b\in N	&\qquad\hbox{Lemma 1.4: $(-1)a=-a$}\cr
a+(-1)(a+b)\in N	&\qquad\hbox{distributive law}\cr
a-(a+b)\in N		&\qquad\hbox{Lemma 1.4}\cr
a+b<a			&\qquad\hbox{definition of less-than}\cr
}$$
